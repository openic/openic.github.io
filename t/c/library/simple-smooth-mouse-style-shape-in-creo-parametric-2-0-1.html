Simple smooth 'Mouse' style shape in Creo Parametric 2.0

A common question with Creo is how create a really simple, smooth 'mouse' like surface in Creo Parametric 2.0 (especially by students).

This is a really simple method of doing so using a combination of swept surfaces and boundary surfaces.

There are many ways of constructing this type of shape & it really depends on how much control you require over the surfaces. For instance I have used two different methods here i.e.

1. A simple uniform sweep feature to create the top surface.

2. A simple swept blend feature to create the top surface, which offers slightly more control / variation of the top surface.

However, you could just as easily create a more complicated set of curves & boundary surfaces to control the top form more accurately. But I wanted to keep things simple in this instance.

I have uploaded 2 full Creo 2.0 Parametric files (with a complete model tree). showing the two variations.
