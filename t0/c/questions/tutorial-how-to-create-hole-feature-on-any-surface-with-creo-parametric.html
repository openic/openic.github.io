    Short tutorial: How to create Hole feature on point with Creo Parametric

    1. Step 1

      Create custom model / shape

    2. Step 2

      Create Point on surface - this point will be reference point for Hole feature

    3. Step 3

      Select Model>>Hole (system create Hole feature on selected point)

    4. Step 4

      Set custom value and shape for your Hole >> Finish

    5. Step 5

      Your Final Hole on any surface
