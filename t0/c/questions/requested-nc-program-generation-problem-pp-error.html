Some day before Miss Aarti Having problem with NC program Generation while she Running the power mill.

The Problem were Identified by me is Missing Duct Processor for Offline program generation.

  • Sourbh 9 months ago

    This problem Generally occurs due to missing Duct processor Configuration,
    Duct Post or processor is generally used for offline program generation for specific CNC Machine.

    while post processor is for Direct NC program generation when the machine is connected to System.

    1. Step 1

      Place the DVD in drive and chose the option for Installing the power mill Either 32 bit or 64 bit.

      But its mostly prefer to Install the 64 bit system.

    2. Step 2

      Accept the License agreement to continue

    3. Step 3

      Do not change the Installing Location

    4. Step 4

      Choose your Short cut option

    5. Step 5

      Select the Item to Install

    6. Step 6

      Select both option under tool path output. This will give you ability to post CNC code from Power mill using Duct post( Offline) and Post processor(Online).

      Also if you need than select the PAF wizard which will help you diagnose the support Issue.

      Optional Power mill Modelling and Data Exchange..!!

      Select Install to Install the software in computer

    7. Step 7

      Congratulation Software is Installed

      Now you are go Normal way for Model machining and Generate the program
      Just you need to Right click on NC program Which is Left hand side tree top after opening the software.

      Select preferences and Open the machine option file, Select the machine option file Specific for A CNC machine.

      Like FANUC Etc.
      Than apply and close.

      After this just Right Click and select the Write Program.
      Than again Right Click on NC program and Click View file to see the Generated program.

      ///Thanks and Enjoy////
