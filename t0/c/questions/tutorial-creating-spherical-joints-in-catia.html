Tutorial : Creating Spherical Joints in CATIA

CATIA mock up spherical kinematics dmu joints digital catia v5
By Rahul Singh Dhari on 18 Jun 05:31 1 answer 108 views 0 comments

In this part I will explain how to create spherical joints in CATIA.

1 answer

  • Rahul Singh Dhari 10 days ago

    Here's the steps to achieve spherical joints.

    1. Step 1

      Click the Spherical Joint icon in the Kinematics Joints toolbar or select Insert -> New Joint ->Spherical from the Menu bar. The Joint Creation: Spherical dialog box appears.

    2. Step 2

      Click New Mechanism. The Mechanism Creation dialog box is displayed.

    3. Step 3

      Select Point 1 in the geometry area. In our example select the ball extremity.
      Select Point 2 in the geometry area. Select a second point (socket extremity).

    4. Step 4

      Click Ok to end the spherical joint creation.
