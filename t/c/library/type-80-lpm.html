This is the Type 80 Chinese tank, a "Low Poly Model".

Made this for a Generals mod, see the 2 pictures...

The diffuse rendered T80, is the 7,99mb SW version and the STEP file, these are somewhat more detailed.

It was too detailed for Generals, so I had to scrap some details (the headlights for example).
The 8,78mb SW file is that low detail version of the T80.

Even though the tank is a low poly model, the tank is made by using blueprints for measurements, so it is correct in scale.
