Tutorial : Creating Roll Curve Motion in CATIA

CATIA mock up kinematics dmu curve digital roll catia v5 motion
By Rahul Singh Dhari on 22 Jun 06:11 1 answer 115 views 0 comments

In this Last part of Kinematics Series I will explain how to create roll curve motion in CATIA.

1 answer

  • Rahul Singh Dhari 6 days ago

    Follow the steps to achieve Roll Curve Joint.

    Attachments

    Roll Curve.7z, 80 KB
    Download

    1. Step 1

      Open the RollCurve_without_kin.CATProduct document.

    2. Step 2

      Select the Roll Curve Joint icon . The Joint Creation: Roll Curve dialog box is displayed.

    3. Step 3

      Click on New mechanism. The Mechanism Creation dialog box is displayed.

    4. Step 4

      Select Curve 1, in our example, select the inner ring in the geometry area. The current selection field is automatically updated with your selection.

      Select Curve 2, for instance select the roller as shown below.

    5. Step 5

      Click Ok to end the roll curve joint creation.

    6. Step 6

      Now, create Roll Curve.2. For this select the Roll Curve Joint icon again.
      The Joint Creation: Roll Curve dialog box is displayed.

    7. Step 7

      Select the outer ring as curve 1 and the roller as curve 2 in the geometry area.
      Click Ok to end the Roll Curve. 2 joint creation.

    8. Step 8

      Click Ok to end the Roll Curve. 2 joint creation.

    9. Step 9

      Now, create Revolute. 3. For this, click the Revolute Joint icon .
      The Joint Creation: Revolute dialog box is displayed.

    10. Step 10

      Select the lines and the planes:

      - Inner ring axis for line 1
      - outer ring axis for line 2

      - zx plane (inner ring)
      - zx plane (outer ring)

    11. Step 11

      Assign the command, for this select the Angle driven check box.

    12. Step 12

      Click Ok to end the revolute joint creation.
      The specification tree is updated.

    13. Step 13

      Click the Fixed Part icon from the Simulation toolbar or select Insert->Fixed Part... from the menu bar. The New Fixed
      Part dialog box is displayed.

    14. Step 14

      Select the inner ring as fixed part either in the geometry area or in the specification tree.
      The specification tree is updated and the mechanism can be simulated.
