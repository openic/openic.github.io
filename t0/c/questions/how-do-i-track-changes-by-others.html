How do i track changes by others?

GrabCAD Workbench revision version management workbench cad grabcad file
By Erez Almog on 02 Jan 13:51 1 answer 273 views 1 comment

If a partner changes a file (edits a part or adds line) how can i see it or track it through GrabCAD?

1 answer

  • Sara Sigel 2 months ago

    Your partner can upload a new version, leave a sketch, or write a comment to let you know about a change. If you have notifications setup, then you'll get an email when any of these happen.

    https://grabcad.com/email/edit

    You can also compare changes between versions, which is very useful. For a visual comparison of what is different from one version to the next, see the steps, below.

    1. Step 1

      Go to the 'Files' view and select your file.

    2. Step 2

      Choose 'Compare' from the 3D Viewer menu.

    3. Step 3

      Select which versions you want to compare

    4. Step 4

      Slide the toggle to see changes disappear or appear.

    5. Step 5

      Note: If there is only one version, this won't be available.
