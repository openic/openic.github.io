      Ctrl-C; Copy
      Ctrl-D; Delete

      Ctrl-F; Fit View
      Ctrl-G; Grip Execute

      Ctrl-J; Edit, Object Display
      Ctrl-L; Format, Layer Settings

      Ctrl-N; File, New

      Ctrl-V; Paste
      Ctrl-W; Application, Gateway
      Ctrl-X; Cut
      Ctrl-Z; Edit, Undo
      Ctrl-Shift-A ; File, Save As

      Ctrl-Shift-O ; Format, Layout, Open
      Ctrl-Shift-C ; View, Curvature Graph
      Ctrl-Shift-D ; Drafting
      Ctrl-Shift-F ; Format, Layout, Fit All Views
      Ctrl-Shift-S ; Tools, Macro, Step
      Ctrl-Shift-G ; Debug Grip
      Ctrl-Shift-H ; High Quality Image Part

    There are missing all the three shortcuts which I use the most in NX 8.5.

    Ctrl + Q = Finish Sketch
    X = Extrude
    Shift + F8 = Normal view to sketch

    The complete list of assigned shortcuts in NX can be found at "Information" - "Custom Menu Bar" - "Shortcut Keys". In "Tools" - "Customize" you can specify your own shortcuts.
