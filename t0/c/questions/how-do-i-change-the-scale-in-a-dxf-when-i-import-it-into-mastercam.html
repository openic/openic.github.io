How do I change the scale in a DXF when I import it into Mastercam?

Mastercam dxf stratocaster neck
By Rob Wiacek on 22 Aug 13:38 3 answers 904 views 0 comments

In many cases, I download a DXF and when I open it in Mastercam the scale is crazy off!
In trying to create an NC file to make Stratocaster neck.
When I open those files the drawings are sooo far off, I will not
Fit in my mill.
Please help. Thanks

3 answers

  • Gordon Long 9 months ago

    In my Mastercam9, the sequence is,
    Xform - scale - all entities - done - point entry origin - then you enter the scale factor and click ok.
    If you use the analyze tool to check the size of a known feature as imported, divide your desired size by the imported size to get the scale factor.
