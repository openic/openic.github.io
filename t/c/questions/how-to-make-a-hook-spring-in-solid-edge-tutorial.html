    It is a basic Solidedge tutorial to make a Hook Spring.

    This tutorial contains two sessions.
    Session 1 : Simple Spring tutorial.
    Session 2 : Hook tutorial.

    Shortcuts:
    CTRL+I - ISO view.
    CTRL+H - Sketch view.
    Right click - Accept a selection.
    Middle click - End command.
    F3 - Lock the sketch plane.

    1. Step 1

      Start Solidedge & also open a new ISO part model.

    2. Step 2

      Select front plane & Lock that plane using F3 .
      Draw the below Sketch - Circle dia 10 mm & 50mm distance from origin.

    3. Step 3

      Select Right plane & start a sketch by locking that plane . Draw a vertical line as below with 100mm length.

    4. Step 4

      Select Helix tool from Solids tab -. Sweep group -> Helix.
      Select Circle and the line (above 2 sketches) and then right click to accept..
      Then give the turns as 4.
      Right click.

    5. Step 5

      Simple Spring is ready.

  • EDISON PHOENIX about 2 years ago

    Session 2: Hook Tutorial .

    Shortcuts:
    CTRL+I - ISO view.
    CTRL+H - Sketch view.
    Right click - Accept a selection.
    Middle click - End command.
    F3 - Lock the sketch plane.

    1. Step 1

      Draw the below sketch in RIGHT plane by locking that plane using F3.
      Finish the sketch by selecting "Select" tool.

    2. Step 2

      Then Select line tool and trace the front plane and then lock it using F3.

    3. Step 3

      Draw the below sketch with endpoints coincidence (with the above sketch) in FRONT plane by locking that plane using F3.
      Finish the sketch by selecting "Select" tool.
      Than Go to Surfacing tab & then select "Cross" tool .

    4. Step 4

      Select first Sketch.
      Right click.
      Select second sketch.
      Right Click.
      Than click "Finish".

    5. Step 5

      Cancel the cross tool.
      And now you can get a curve like as Hook's profile path.
      We can use this curve as a profile path of the Hooks.(Using Sweep)

    6. Step 6

      Click "Project to sketch" tool and then trace the mentioned plane and lock using F3.

    7. Step 7

      Project the circle as given below.
      We can use this sketch as the cross section of Hook. (Using Sweep)

    8. Step 8

      Click Sweep tool. Then click "OK".

    9. Step 9

      Select the curve chain as profile path of the Sweep as given below.
      Right Click.
      Select the projected circle as the cross section of the Hook-Sweep.
      Right click & finish.

    10. Step 10

      Now you can get the hook model.
      Then fillet the edges using "Round" tool.
      Now one side hook is made.

    11. Step 11

      Repeat these above steps once again to get the next end hook.
      Finally you can get the HOOK SPRING as shown below.
